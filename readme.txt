Packages: (Install Packages using terminal or command prompt using the pip command)
- Install pygame using "pip install pygame"
- Install numpy using "pip install numpy"
- Install requests using "pip install requests"

Overview:
This application is a simple version of the pokemon game, which allows you to search up information about the first generation pokemon
and also create a party and battle a random opononent party to test out pokemon you want to use. This application will allow pokemon players
to test out pokemon they want to use competitively.
